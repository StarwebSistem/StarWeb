Política de Privacidade

Esta Política de Privacidade descreve como coletamos, usamos e protegemos as informações dos usuários do aplicativo Meu Evento, disponível para dispositivos móveis Android.

1. Informações Coletadas

Ao utilizar o aplicativo, coletamos as seguintes informações pessoais:

  • Nome completo
  • Endereço de e-mail
  • Telefone
  • Data de nascimento

Esses dados são fornecidos voluntariamente pelo usuário no momento do cadastro e são essenciais para a gestão de eventos e participantes.

2. Uso das Informações

As informações coletadas são utilizadas para os seguintes fins:

  • Cadastro e gerenciamento de eventos
  • Controle e identificação de participantes
  • Envio de comunicações relacionadas ao evento
  • Relatórios e organização do evento
  • Processamento de pagamentos via Pix (dados bancários não são armazenados)

3. Armazenamento de Dados

Todos os dados são armazenados de forma segura utilizando os serviços da AWS (Amazon Web Services). Tomamos medidas técnicas e administrativas para proteger suas informações contra acesso não autorizado, alteração ou destruição.

4. Compartilhamento de Dados

Não compartilhamos suas informações pessoais com terceiros, exceto quando exigido por lei ou com seu consentimento expresso.

5. Pagamentos

Os pagamentos no aplicativo são realizados exclusivamente via Pix. O app não armazena dados bancários ou informações sensíveis de pagamento.

6. Direitos do Usuário

Você tem o direito de acessar, corrigir ou solicitar a exclusão dos seus dados pessoais a qualquer momento. Para isso, entre em contato conosco pelo e-mail abaixo.

Informações sobre idade e bebidas alcoólicas

Coletamos a data de nascimento do usuário para verificar a idade mínima legal exigida para compra de bebidas alcoólicas. Essas informações são usadas exclusivamente para esse controle e seguem todas as normas da Lei Geral de Proteção de Dados (LGPD).

7. Alterações nesta Política

Podemos atualizar esta Política de Privacidade periodicamente. Recomendamos que o usuário consulte esta página regularmente para estar ciente de eventuais mudanças.

8. Contato

Se tiver dúvidas ou quiser exercer seus direitos relacionados à privacidade dos seus dados, entre em contato conosco:

E-mail de suporte: suporte@meuevento.app
